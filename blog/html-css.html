Margins, Borders & Padding

What is the difference?
What are some examples?

If i was to describe what each component is to someone who is not tech save i would say if all these parts were your property, the margin is like the footpath around your house. the border is like your fence line. the padding is your garden, and the content is your house.

Margin

I am going to break down each of these parts individually starting from the outermost component.
The margin sets a space around the edge of the border as a buffer zone, it acts like an invisible force bubble. This cannot be visually styled outside of sizing of the space it takes.
The margin can have a blanket value or different values for all sides being top, left, bottom, and right. This value can be in different forms, px (pixel), and % are the more common uses.
Below is an example of the code in an inline format:

<p style="margin-left: 20%; margin-right: 20%; background-color: rgb(112, 0, 0);">this is an example</p>

this is an example


Border

The border is the outline of the "box", it is the visual representation of the outer layer of your content. you can style this in many ways.
An example is using a "border-radius: 50%;" to give your "box" a circle shape. you can also colour your border.
Like mentioned above for margin, you can have different values for each side or a general blanket value, this also applies for padding.
Below is an example of the code in an inline format:

<p style="border: 5px solid rgb(112, 0, 0);">this is an example</p>

this is an example


Padding

The padding around content is the space between the content and the border. Just like the margin, this is invisible and cannot be styled outside of its size.
Below is an example of the code in an inline format:

<p style="padding: 20px; border: 2px solid white; background-color: rgb(112, 0, 0);">this is an example</p>
